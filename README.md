# Progress Navigator

A simple task and progress tracking application built with React and Vite.

## Features

- Create and manage tasks
- Track completion progress
- Clean and minimal UI
- Fast performance using Vite
- Local development with hot reload

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Installation

Clone the repository:

git clone https://github.com/ryanrt606-del/progress-navigator.git

Go into the project:

cd progress-navigator

Install dependencies:

npm install

Run the development server:

npm run dev

Open in browser:

http://localhost:8080

## Build for production

npm run build

Preview production build:

npm run preview
